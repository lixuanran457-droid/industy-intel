# 趋势判断标准

## 一、判断标准（什么信息对我有益）

### 🔴 高价值信息
| 特征 | 说明 | 示例 |
|------|------|------|
| 变现模式创新 | 新的付费/赚钱方式 | "某创作者通过AI定制内容月入$10万" |
| 平台政策变化 | 影响整个行业 | "OnlyFans调整分成比例" |
| 技术突破 | 工具/玩法升级 | "AI换脸技术成熟，降低创作门槛" |
| 新平台崛起 | 流量重新分配 | "某新平台三个月获取百万用户" |

### 🟡 中等价值信息
| 特征 | 说明 | 示例 |
|------|------|------|
| 运营技巧 | 可借鉴的策略 | "如何通过TikTok引流到成人平台" |
| 用户行为变化 | 需求趋势 | "年轻用户更偏好匿名/隐私" |
| 内容形式创新 | 新内容形态 | "互动视频内容的崛起" |

### 🟢 参考价值
| 特征 | 说明 | 示例 |
|------|------|------|
| 行业八卦 | 了解圈子文化 | "某知名创作者争议事件" |
| 技术讨论 | 了解技术趋势 | "成人网站的推荐算法讨论" |

## 二、趋势信号识别

### 上升趋势信号
1. **多个平台同时出现** — 同一个玩法在多个地方出现
2. **搜索量持续上升** — Google Trends显示上升曲线
3. **头部账号跟进** — 大V/知名创作者开始使用
4. **资本进入** — 投资机构开始投资相关项目
5. **媒体报道增加** — 主流媒体开始报道

### 衰退趋势信号
1. **平台政策收紧** — 越来越多限制
2. **用户抱怨增加** — 社区负面反馈
3. **创作者流失** — 头部创作者离开
4. **搜索量下降** — 相关词搜索量下滑

## 三、竞品价值评估

### 评估维度
| 维度 | 评估问题 | 权重 |
|------|----------|------|
| 流量 | 访问量多少？增长趋势？ | 25% |
| 商业模式 | 怎么赚钱？可持续吗？ | 25% |
| 用户体验 | 产品好不好用？ | 20% |
| 差异化 | 有什么独特之处？ | 15% |
| 团队 | 有没有背景？ | 15% |

### 竞品分类
```
┌─────────────────────────────────────────────┐
│                  高价值竞品                  │
│  流量大 + 商业模式清晰 + 体验好              │
│  → 深度研究，借鉴优点                       │
├─────────────────────────────────────────────┤
│                  观察竞品                   │
│  有流量 + 商业模式待验证                     │
│  → 持续监控，看趋势                         │
├─────────────────────────────────────────────┤
│                  警惕竞品                   │
│  新入场 + 大力补贴 + 不可持续                │
│  → 关注市场反应，不盲目模仿                 │
└─────────────────────────────────────────────┘
```

## 四、商机判断

### 判断问题
1. 这个玩法我能复制吗？
2. 我有什么优势去做这个？
3. 投入产出比是多少？
4. 风险有多大？
5. 时机对吗？（太早还是太晚？）

### 商机优先级
| 优先级 | 条件 |
|--------|------|
| 🔴 立即行动 | 能复制 + 有优势 + 时机对 |
| 🟡 研究跟进 | 有前景 + 时机未到 |
| 🟢 了解即可 | 纯信息 + 不适合当前 |

## 五、每周关键词评估

### 热度判断标准
| 指标 | 判断方法 |
|------|----------|
| 搜索量 | Google Trends / 百度指数 |
| 社交讨论量 | Reddit帖子数 / 微博话题 |
| 竞品覆盖 | 有多少竞品在追这个 |
| 变现案例 | 有没有人靠这个赚钱 |

### 关键词价值评估
```
高价值关键词特征：
✓ 搜索量上升
✓ 多平台同时出现
✓ 有商业化案例
✓ 与我的业务相关
✓ 还没被大平台垄断

低价值关键词特征：
✗ 纯娱乐/猎奇
✗ 没有商业价值
✗ 违禁/高风险
✗ 已经是红海
```
